## FIRE-Specific Value

This design helps FIRE users by:

1. **Passive vs Active Income Split** - Track progress toward not needing a job
2. **Investment Allocation** - See how much goes toward wealth building
3. **Money Flow Visualization** - Understand the complete picture
4. **Asset Activity** - Know which accounts are working hardest
5. **Net Receiver/Sender** - Identify wealth accumulation patterns
